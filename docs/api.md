# 返回体格式

若无特殊说明，返回体均为`application/json`格式：

```json
{
  "code": JsonResponsecode,
  "info": info,
  "data": {
    /* 返回体 */
  }
}
```

# 通用返回值

为了节约篇幅，通用的异常返回值如下

| HTTP状态码 | 说明                                       |
| ---------- | ------------------------------------------ |
| 400        | 通用错误，具体错误由JsonResponse状态码决定 |
| 401        | 鉴权失败：用户未登陆                       |
| 405        | 请求方法不允许                             |
| 500        | 服务端其他内部错误                         |
